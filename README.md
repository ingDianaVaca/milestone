# Milestone 1: Project setup and git basics

## Prerequisites

Make sure you have Java installed (If Java is not installed, follow the installation steps on the official Java website)

```bash
  java –version
```
Make sure you have Maven installed (If Maven is not installed, follow the installation steps on the official Maven website)

```bash
  mvn −version
```

## Installation
To facilitate the project creation process, we will use the “Spring Initializr” tool where we will make sure to check:

- Project: Maven

And add the dependency: 
- Spring Web

Generate the zip file and open it with the IDE of your choice.
